## About

A clean, minimal portfolio showcasing selected product work — from design through full-stack build and deployment. Built and shipped solo.

## Featured Projects

- **SiteLog** — A B2B site-reporting app for construction teams to track daily reports, issues, and material requests. Multi-tenant, role-based, mobile + desktop.
- **Paylio** — A link-based payment tool that lets social-commerce sellers get paid without a storefront.
- **MailMint** — An AI-powered platform for generating structured, conversion-focused email sequences.
